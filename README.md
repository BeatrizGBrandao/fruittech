# FruitTech - Projeto de Monitoramento de Transporte de Frutas e Hortaliças

Bem-vindo ao repositório do projeto de Monitoramento de Transporte de Frutas e Hortaliças. Este projeto tem como objetivo desenvolver um sistema de monitoramento que ajude agricultores a reduzir perdas durante o transporte de hortaliças e frutas. O sistema utiliza sensores Arduino de temperatura e umidade em combinação com software de monitoramento em tempo real.

## Funcionalidades Principais

- Monitoramento em tempo real da temperatura e umidade dos produtos durante o transporte, por meio de sensores Arduino.
- Geração de alertas em caso de condições fora dos padrões.
- Histórico de dados para análise posterior.

## Estrutura do Repositório

- `/backend`: Contém o código fonte do backend do sistema.
- `/frontend`: Contém o código fonte do frontend da aplicação web.
- `/docs`: Documentação relacionada ao projeto.
